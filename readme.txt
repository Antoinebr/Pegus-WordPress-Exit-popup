Pegus exit popup allow you to easily add an exit popup to your site. 

== Description ==

With Pegus you can push a last message before your visitors leave.
You can chose betwen 4 modules :


1. A standard message.
1. Ask your visitors to share your content on (Facebook, Twitter or Google +).
1. Ask your visitors to signup to your newsletter (MailChimp) with an AJAX form.
1. Use the custom HTML module to fully customize the exit popup.

Note that you can set a minimum time before the popup execution, and set a maximum number of executions.


== Installation ==

This section describes how to install the plugin and get it working.

1. Unzip the folder
1. Upload pegus-exit-popup to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= Can i limit the number of popup execution ? =

Yes, you can limit that by putting the max number of execution  on the settings.
